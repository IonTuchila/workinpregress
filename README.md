# Locations App
An Android App where you can se some location with some datas about it , it was built with mvvm, dagger-hilt, livedata,coroutines, jetpack libraries...
 

# Features

1- Kotlin  
2- MVVM design pattern  
3- Retrofit2   
4- ViewBinding    
5- Coroutine  
6- Picasso
7- LiveData  
8- Navigation Graph
9- Dagger-Hilt
